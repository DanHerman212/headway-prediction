# NYC Subway Headway Prediction

Real-time headway prediction for A, C, E subway lines at West 4th Street-Washington Square station.

---
This project has been suspended.  I was unable to achieve reasonable accuracy to create a passenger application to predict "minutes until the next train".  The real time event driven architecture included in this repo worked really well with google cloud dataflow. 

This was a good learning experience to build a full stack data app.  The prediction task was too difficult to execute, but the surrounding data infrastructure worked pretty well.
